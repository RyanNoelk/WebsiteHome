Sudoku Web App

This is a simple Django sudoku web app. It contains two core pieces. Python classes that generate, solve, and check any x by x (square) sudoku puzzle. UI constructed with Bootstrap and jQuery. The app will generate random puzzles and store them within a DB to be served to the user. The libraries can be used in a standalone mode to generate or solve any square sudoku puzzle!
